The following is a checklist of things to do in Kingdoms of Amalur: Re-Reckoning, mostly side quests and tasks. Repeatable tasks do not have a checkbox because they are not "completeable". Main quests and faction quests are not included as they are fairly linear. It is a fork of this guide which I found IMMENSELY helpful (and others like it) with my journey (read: painful slogging) through the soulsborne games. Re-Reckoning: Fate Edition went on sale, and I decided to make this to help myself make sure I got (close?) to 100% completion. Quests are split up between areas, based on this guide, here. I will continue to add to this to include the three DLC. Links are provided to amalur.wikia.com where available.

Warning: Contains Spoilers!

Help

Is this a guide?

No. This is simply a checklist of side quests, etc. to help make sure you get 100% completion (but this is not a 100% completion guide). I made it in part to help make sure I did all quests in a particular area to coincide with the anti-overlevelling and area cap guide, which you can read more about here.

Can I use this for multiple characters?

Yup, use the profile selector and buttons at the top right of the page to setup multiple characters.

How do I help with feedback, corrections, suggestions or additions?

Create a new issue or fork and submit a pull request. I might get to it.

How does the checklist status get saved?

It gets saved to the local storage in your browser and not to any database or API. So if you are clearing your browser cache make sure not to delete the site data or your checklists will get reset.
